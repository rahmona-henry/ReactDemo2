

# counterGreeter

A simple dynamic name greeter app that is built using React.

## how

### install

```
git clone https://github.com/rahmona-henry/counterGreeter
cd
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
